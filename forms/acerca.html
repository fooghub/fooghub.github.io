fooghub.github.io: Es un sitio Web público, personalizado, de aspecto flexible y contenido dinámico. Con información sobre algunos proyectos de software publicados en la plataforma de desarrollo corporativo GitHub, por Fooghub.

Funciona en cualquier dispositivo con un navegador gráfico actualizado, aunque está optimizado para las dimensiones y resolución visual de la pantalla de los ordenadores personales.

Programado con los lenguajes: HTML5 para su estructura básica, CSS3 para el diseño gráfico y JavaScript1 estándar para interactuar con sus funciones o conseguir la presentación de contenido variable (AJAX).

Está hospedado en Github Pages, un servicio de alojamiento gratuito, ofrecido por GitHub para páginas Web estáticas2 de sus usuarios.

  1. JavaScript de otros autores:
    — Algunas funciones de la Librería Modernizr para comprobar si la versión del navegador del usuario es adecuada para la correcta visualización de esta página.
    — El script browser.js facilita la detección de algunas características del dispositivo y nagegador del usuario.
    — El ofuscador de Tim Williams, permite encriptar en el código las direcciones de correo electrónico para evitar el spam.
    — Librería smoothScroll de Alice Lieutier. Consigue un efecto de desplazamiento suave entre contenidos.
    — Contamos con la ayuda de algunas idéas de la creciente comunidad Stack Overflow y de la Documentación web de MDN - Mozilla.
  2. Github Pages:
    — Es un servicio de alojamiento web de tipo estático: No tiene acceso a una base de datos, tampoco dispone de soporte para el procesado previo en el servidor (PHP, por ejemplo). Pero integra Jekyll, un generador para la escritura, diseño y publicación de blogs.
